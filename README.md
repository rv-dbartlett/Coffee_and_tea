# Coffee_and_tea

This is an example application and tests to explain how to write unit, integrations, and e2e tests.

## Libraries used
- [express](https://expressjs.com/en/4x/api.html)
- [mocha](https://mochajs.org/)
- [sinon](https://sinonjs.org/releases/v9.0.2/)
- [chai](https://www.chaijs.com/guide/)
- [chai-http](https://www.chaijs.com/plugins/chai-http/)
- [sinon-chai](https://www.chaijs.com/plugins/chai-sinon/)
- [selenium-webdriver](https://www.selenium.dev/selenium/docs/api/javascript/index.html)
